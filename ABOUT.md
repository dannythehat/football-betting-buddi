# About FootballBettingBuddi

FootballBettingBuddi is a user-first platform that blends numeric probability models with GPT reasoning to recommend high-value bets across all major markets (1X2, O/U goals, BTTS, Asian handicap, corners, cards, etc.).  
It aims to give users **clarity** (probabilities, EV, confidence colors) and **simplicity** (a single **AI Smart Bet** button per game).

- **Free Tier:** Core features, daily Smart Bets (limited).
- **Premium:** Extended tips, detailed confidence, early alerts, API access.
- **Principles:** Transparency, historical tracking, and user-controlled market focus.
