# Roadmap

## Phase 0 — Foundation (THIS STAGE)
- Repo structure and documentation (no code)
- Identity: README, ABOUT, VISION, ROADMAP, STATUS, MASTER_SPEC

## Phase 1 — MVP (Docs → Prototype)
- Vanilla JS UI (dark theme) with fixtures + AI Smart Bet buttons
- Backend jobs (daily) produce 2–5 Smart Bets (JSON cache)
- Market support: 1X2, O/U 2.5, BTTS, AH, Corners O/U, Cards O/U
- Anonymous dashboard via localStorage; export CSV; basic P/L auto-update

## Phase 2 — Odds & EV
- Bookmaker odds ingestion; Expected Value based on live odds
- Confidence visuals + explanations per prediction

## Phase 3 — Accounts & Premium
- Auth, Stripe subscriptions, Premium-only daily tips and analytics

## Phase 4 — API & Integrations
- Partner API for fixtures, predictions, Smart Bets
- Alerts & notifications (injuries, line moves)

## Phase 5 — Scale
- Postgres migration, React/Vite rewrite, mobile app, white-label options
