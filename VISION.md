## Target Audience

### Primary Users
1. **Casual Bettors** (60% of users)
   - Bet occasionally on weekends
   - Want simple, trustworthy tips
   - Value transparency and education
   - Free tier users

2. **Regular Bettors** (30% of users)
   - Bet multiple times per week
   - Seek value and EV opportunities
   - Track performance and ROI
   - Premium subscribers

3. **Professional Bettors** (10% of users)
   - Bet daily across multiple markets
   - Use data for arbitrage and value betting
   - Need API access and bulk data
   - Premium + API subscribers

### Secondary Users
- **Betting Syndicates**: Group subscriptions, API access

- **Data Analysts**: Historical data for research
- **Betting Platforms**: White-label solutions
